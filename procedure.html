Gebruikersinstructie

Dit gebruikersinstructie omschrijft het maken en gebruiken van standaardoplossingen in TOPdesk. Hierin worden de volgende dingen beschreven:

• Hoe je een standaardoplossing aanmaakt.

• Hoe standaardoplossingen gebruikt kunnen worden bij het invoeren van eerstelijnsmeldingen.

De gebruikersinstructie is bedoeld voor medewerkers van de servicedesk bij AMERSict.

Hoe je een standaardoplossing aanmaakt

1. Ga in het home scherm naar topdeskmenu, nieuw, meldingbeheer, standaardoplossing.

2. Vul het veld categorie in

3. Vul het veld Subcategorie in.

4. Sla de standaardoplossing op.

Hoe standaardoplossingen gebruikt kunnen worden bij het invoeren van eerstelijnsmeldingen.

1. Maak een eerstelijns melding aan

2. Vul de categorie in.

3. Vul de subcategorie in.

4. Als er een standaardoplossing is voor de subcategorie van een melding of voor een trefwoord in het verzoek gaat er een lampje branden, klik op het lampje.

5. Kies de juiste standaardoplossing en voeg deze toe aan het actieveld van de melding.

6. Vul de overige velden van de melding in en sluit de melding af.

User instruction

This user instruction describes the making of a survey in TOPdesk. This instruction gives information about the following things:

• The making of a survey.

• Adding questions and possible answers.

• Linking a survey to a campaign

This instruction is intended for English-speaking employees.

The making of a survey.

1. Choose TOPdesk-menu, modules, enquetes.

2. Press the button Enquete in the box ‘nieuw’ on the Enquetebeheermodulepagina.

3. Give the enquete a title.

4. Save the enqeute.

Adding questions and possible answers.

1. Press on the button ‘nieuwe vraag’ to register a new question.

2. Fill in ‘Korte omschrijving’ a short description.

3. Select by ‘Vraagtype’ if it is a open question ‘Open vraag’ or if it is a multiple choice question ‘Meerkeuzevraag’.

4. Select the box ‘Vraag met meerdere antwoorden mogelijk’ if the question have multiple answers possible.

5. Select the box ‘Verplicht’ if the question is mandatory.

6. Select by the box ‘Vraagcategorie’ the category of the question.

7. Fill in the field ‘Vraag’, the question you want to ask.

8. Press on the save

9. Press at the bottom of the screen on ‘Nieuwe antwoordmogelijkheid’ to register an new answer.

10. Fill in the possible answers.

11. Save the question

If you want to add more questions repeat the above steps.

Press the button ‘voorbeeld’ if you want to see a example of how the survey would look for the user.
